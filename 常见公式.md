## 有重复元素的排列
 如果总共有 n 个元素，其中第一类元素有 $n_1$ 个相同的，第二类有 $n_2$ 个相同的，...，第 k 类有 $n_k$ 个相同的（$n_1 + n_2 + ... + n_k = n$），则排列数为：
$\frac{n!}{n_1! \times n_2! \times ... \times n_k!}$
*   **例子**: 单词 “SUCCESS” 中字母的排列数。总字母数 n=7，其中 S 有 3 个，C 有 2 个，U 和 E 各有 1 个。排列数为 $\frac{7!}{3! \times 2! \times 1! \times 1!} = 420$。


## 可重复组合 / 隔板法
 从 n 个不同元素中**可重复地**选取 m 个元素（每种元素可以选多次），不考虑顺序，方案数为：
$\binom{n + m - 1}{m}$ 或 $\binom{n + m - 1}{n - 1}$
*   **理解**: 这个问题等价于求方程 $x_1 + x_2 + ... + x_n = m$ 的**非负整数解**的个数。经典的 **“隔板法”** 模型。
*   **例子**: 将 7 个相同的苹果分给 3 个小朋友，每人可以有 0 个，有 $\binom{3+7-1}{7} = \binom{9}{7} = 36$ 种分法。


## 分组组合
将 n 个不同的元素分成 **k 个组**，每组的人数分别为 $n_1, n_2, ..., n_k$ ($n_1+n_2+...+n_k = n$)，且组之间**无顺序区分**（例如只是分组，不编号），则方案数为：
$\frac{n!}{n_1! \times n_2! \times ... \times n_k!}$
*   **注意**: 如果组是**有区别的**（例如标号为第1组、第2组），则方案数要再乘以 $k!$，即 $n!$。
*   **例子**: 10 个人分成 3 人、3 人、4 人的三个**无标签**小组，分法为 $\frac{10!}{3!3!4!}$。如果是三个**有标签**（如 A组/B组/C组）的小组，分法为 $\frac{10!}{3!3!4!} \times 3!$。


## 卡特兰数
极其重要的数列，解决许多**括号匹配、二叉树形态、出栈序列**等问题。
$C_n = \frac{1}{n+1}\binom{2n}{n} = \binom{2n}{n} - \binom{2n}{n-1}$
*   前几项: 1, 1, 2, 5, 14, 42, ...
*   **应用场景**:
    *   n 对括号的合法匹配序列数
    *   n 个节点的不同形态的二叉树数量
    *   一个栈的进栈序列为 1,2,...,n，不同的出栈序列数
    *   在网格中从 (0,0) 走到 (n,n) 且不越过对角线的路径数


## 错位排列
n 个元素的排列，每个元素都不在自己原来的位置上。
$D(n) = n! \sum_{k=0}^{n} \frac{(-1)^k}{k!}$
*   递推公式: $D(n) = (n-1)[D(n-1) + D(n-2)]$，其中 D(1)=0, D(2)=1。


## 两序列错位相插

要将 **A、B** 两个序列**合并**成一个新的长度为 `m + n` 的序列，规则是：
1.  必须**保持序列 A 内部元素的原有相对顺序**。
2.  必须**保持序列 B 内部元素的原有相对顺序**。
3.  序列 A 和序列 B 的元素可以任意交错。

问题等价于：从这 `m + n` 个位置中，**选出 `m` 个位置**用来放置序列 A 的元素。

故总方案数为：
**$C(m + n, m) = C(m + n, n) = \frac{(m+n)!}{m! \cdot n!}$**
